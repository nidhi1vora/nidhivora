# Nidhi Vora — Portfolio

Personal portfolio site for Nidhi Vora, a Computational Finance & Risk Management and CS student. It's a single-page, sidebar-navigation site covering an about/intro, project highlights (quant finance & ML projects), internship experience, and contact info. Built with plain HTML/CSS/JS — no framework, no build step.

## Key Features

- Dark-themed, sidebar navigation layout with smooth scrolling and scroll-spy active-link highlighting
- Sections for About, Projects, Internship, Resume, and Contact
- Fully responsive (sidebar collapses on mobile)
- Project cards summarizing quant finance / ML work (spam classifier, stock recommendation system, options pricing model)

## Tech Stack

- HTML5, CSS3 (custom properties for theming), vanilla JavaScript
- Deployed via GitHub Pages

## Prerequisites

- None — it's static HTML/CSS/JS, no build tools or dependencies required

## Getting Started / How to Run

```bash
git clone https://github.com/nidhi1vora/nidhivora.git
cd nidhivora
# Open index.html directly in a browser, or serve it locally:
python -m http.server 8000
```

Then visit `http://localhost:8000`.

### Deploying via GitHub Pages

1. Go to **Settings → Pages**
2. Under **Source**, select **Deploy from a branch**
3. Choose **main** and **/ (root)**
4. The site will be live at `https://nidhi1vora.github.io/nidhivora/`
